## 👁 Про мене

Привіт. Я **Станіслав**, розробник власних Telegram-проєктів та систем під брендом **RotKranz**.

Я працюю над:
- Telegram-ботами на Python
- ігровими механіками та автоматизацією
- VPS / Linux серверами
- Flask / backend частиною сайтів
- SQLite / PostgreSQL
- GitHub-проєктами та деплоєм

Мені подобається будувати не просто код, а **цілу екосистему**: бот, сайт, міні-апп, економіка, статистика, ігрові системи та спільнота навколо цього.
